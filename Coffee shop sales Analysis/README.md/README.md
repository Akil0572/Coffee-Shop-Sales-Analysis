# Coffee Shop Sales Analysis

## 📌 Objective
The goal of this project is to analyze coffee shop sales data to understand sales trends, top-selling products, and customer behavior, helping businesses make data-driven decisions.

## 🛠 Tools & Technologies
- Microsoft Excel
- Pivot Tables
- KPIs
- Data Cleaning Techniques

## 🔄 Data Preparation
- Removed duplicate records
- Handled missing/null values
- Standardized data formats for analysis

## 📊 Analysis & Dashboard
- Created KPIs to track total sales and performance
- Used Pivot Tables to analyze sales by product, time, and category
- Built an interactive Excel dashboard for visualization

## 📈 Key Business Insights
- Weekends generate higher revenue than weekdays
- Coffee products contribute the majority of total sales
- Peak sales occur during evening hours

## 📁 Project Structure
- data/ – Dataset used for analysis
- dashboard/ – Dashboard screenshots
- insights/ – Business insights documentation

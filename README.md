# ☕ Coffee Shop Sales Analysis

## 📌 Project Objective
The objective of this project is to analyze coffee shop sales data to identify sales trends, top-performing products, and customer behavior, enabling data-driven business decisions.

---

## 🛠 Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- KPIs
- Data Cleaning Techniques

---

## 🔄 Data Cleaning & Preparation
The following steps were performed to prepare the dataset for analysis:
- Removed duplicate records
- Handled missing and null values
- Standardized data formats
- Ensured data consistency for accurate analysis

---

## 📊 Analysis & Dashboard
- Created key performance indicators (KPIs) to track overall sales performance
- Used Pivot Tables to analyze:
  - Product-wise sales
  - Time-based sales trends
  - Category performance
- Built an interactive Excel dashboard for visualization and reporting

---

## 📈 Key Business Insights
- Weekend sales are higher compared to weekdays
- Coffee products contribute the majority of total revenue
- Peak sales occur during evening hours
- Certain low-performing products can be optimized or promoted
